-npm install

-npm run serve

-port 3000(ako je potrebno promijeniti port, to se moze uraditi u frontend-isa/vue.config.js)

-ako je u frontend aplikaciji potrebno promijeniti port bekend apliakcije, moze u frontend-isa/src/main.js
